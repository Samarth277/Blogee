### Run Locally

To run **Blogee** locally, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/Samarth277/Blogee.git
```

Windows:

```bash
git clone https://github.com/Samarth277/Blogee.git
```
